# Bookinsanity

Body-text font for content in The Codex — a community remake of **Bookmania**,
the body face of the D&D 5th Edition core books.

- Created by Solbera (with fixes by Ryrok and Ners), obtained from
  <https://github.com/jonathonf/solbera-dnd-fonts>
- License: **CC-BY-SA-4.0** — attribution above satisfies the BY clause;
  redistribute any modified font files under the same license.

Loaded via `@font-face` in `src/styles.css` as `--font-body`.
